# School Management API

## Features
- Add School API
- List Schools sorted by distance
- Update School API

## Tech Stack
- Node.js
- Express.js
- MySQL

## Endpoints

### Add School
POST /addSchool

### List Schools
GET /listSchools?latitude=17.385&longitude=78.4867

### Update School
PUT /updateSchool/:id

## Setup
1. npm install
2. Setup MySQL
3. node app.js
